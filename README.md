# My Timetable Bot

Telegram-бот для студентов на Python.

## Что умеет бот

- показывать пары на сегодня, завтра, неделю и месяц
- показывать задачи на дату, за период, по категории и все задачи
- показывать полный календарь: пары + задачи вместе
- добавлять, редактировать и удалять пары
- добавлять, редактировать и удалять задачи
- создавать, редактировать и удалять категории
- хранить параметры напоминаний и повторов

## Основные разделы

- `📊 Мой день`
- `📅 Расписание`
- `📝 Мои задачи`
- `📆 Полный календарь`
- `🔧 Управление`
- `ℹ️ Помощь`

## Команды

- `/start` — запустить бота
- `/help` — показать справку

## Установка локально

1. Клонируй репозиторий
2. Установи зависимости:
pip install -r requirements.txt
3. Добавь токен в .env:
BOT_TOKEN=твой_токен
4. Запусти бота:
python bot.py

## Хранение данных

Бот использует SQLite базу данных timetable.db.

В базе данных хранятся:
- пары
- задачи
- категории

## Деплой на Railway

1. Залей проект в GitHub
2. Создай проект на Railway
3. Подключи GitHub репозиторий
4. В Railway добавь переменную окружения:
BOT_TOKEN = твой токен от BotFather
5. Railway сам запустит bot.py

## Важно

SQLite хорошо подходит для учебного проекта и курсовой работы.
